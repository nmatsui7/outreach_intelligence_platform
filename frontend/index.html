Integrations

Read-only architecture overview

The platform is connector-agnostic. AI features use the internal service layer and do not depend directly on Salesforce, Google Drive, Gmail, or any specific external system. The current MVP uses local demo storage only. Future connectors could provide organization data, contact data, interaction history, and related documents through the same internal API.

Active Local Connector

Local Demo CRM

Active

Uses local project data only. Stores organizations, contacts, interactions, AI assessments, drafts, and demo outbox records.

Safety: No network access. No credentials required. All data stays in the project directory.
